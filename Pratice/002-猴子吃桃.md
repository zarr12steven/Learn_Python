# 猴子吃桃

===

```
題目：

一隻猴子第一天摘下若干個桃子，並吃了一半，感覺還吃不過癮，於是又多吃了一個，
第二天早上，又將剩下的桃子吃掉一半，又多吃了一個，
之後每天早上都吃了前一天剩下的一半又零一個。

請問到了第10天早上想再吃的時候，卻發現只剩下一個桃子，求第一天共摘了幾個桃子？
```

**解題思路**

```
用遞迴的方式，把日期從 9 .. 0 依序遞減

第十天剩一個桃子 p = 1

每天都吃前一天剩下的一半又一個桃子 p = ( p + 1 ) * 2 

```
 
**代碼**

```
#!/usr/bin/env python3
# -*- coding:utf-8 -*-

p = 1
print("第十天吃之前就剩下一個桃子")
for i in range(9, 0, -1):
    p = (p + 1) * 2
    print("第{}天吃了之前還有{}個桃子".format(i,p))
print("第一天共摘了{}個桃子".format(p))

```
